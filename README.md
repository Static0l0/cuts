# ✂️ cuts

A simple, clean URL shortener — built with Flask and deployed on Vercel.

🔗 **Live at:** [cuts.my.id](https://www.cuts.my.id)

---

## About

**cuts** is a minimal URL shortener focused on simplicity — both in design and usage. Paste a long URL, get a short one. That's it.

---

## Tech Stack

| Layer      | Technology              |
|------------|-------------------------|
| Backend    | Python + Flask          |
| Frontend   | HTML + CSS              |
| Database   | [Neon](https://neon.tech) (PostgreSQL) |
| Deployment | [Vercel](https://vercel.com) |
